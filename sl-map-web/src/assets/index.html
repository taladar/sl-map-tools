Render Second Life maps from a grid rectangle or a USB notecard.

Output settings

Enabling "Fill missing regions" significantly increases render time because each region inside an existing map tile has to be checked individually against the upstream server.

Region names and grid coordinates are only drawn when each region is rendered large enough for the text to fit and the render covers a limited number of regions (roughly a 32×32 area) — at smaller sizes or larger areas the names will not fit and resolving every name would be too slow, so they are skipped. The rectangle outline is always drawn when enabled.
